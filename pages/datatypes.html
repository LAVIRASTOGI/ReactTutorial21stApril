Data Types in JavaScript

JavaScript has eight basic data types that can be divided into two main categories: primitive and reference types.

Understanding data types is fundamental to programming in JavaScript as it affects how values are stored, compared, and manipulated.

Primitive Data Types

Primitive values are immutable (cannot be changed) and are passed by value. 

// String
const name = "Alice";

// Number
const age = 30;
const pi = 3.14159;

// Boolean
const isActive = true;

// Undefined
let username;  // Value is undefined

// Null
const data = null;

// Symbol (ES6)
const id = Symbol('id');

// BigInt (ES2020)
const bigNumber = 9007199254740991n;

Reference Data Types

Reference types are mutable and are passed by reference. 

// Object
const person = {
  name: "Bob",
  age: 25
};

// Array (which is a type of object)
const colors = ["red", "green", "blue"];

// Function (which is a type of object)
function greet() {
  return "Hello!";
}

// Date (which is a type of object)
const today = new Date();

Type Checking

You can check the type of a value using the typeof operator: 

console.log(typeof "Hello");  // "string"
console.log(typeof 42);       // "number"
console.log(typeof true);     // "boolean"
console.log(typeof undefined);// "undefined"
console.log(typeof null);     // "object" (this is a known JavaScript bug)
console.log(typeof Symbol()); // "symbol"
console.log(typeof 10n);      // "bigint"
console.log(typeof {});       // "object"
console.log(typeof []);       // "object" (arrays are objects)
console.log(typeof function(){});// "function"

Type Conversion

JavaScript allows values to be converted between types:

// String to Number
console.log(Number("42"));     // 42
console.log(parseInt("42"));   // 42
console.log(parseFloat("3.14"));// 3.14
console.log(+"42");            // 42 (unary plus)

// Number to String
console.log(String(42));       // "42"
console.log((42).toString());  // "42"
console.log(42 + "");          // "42" (implicit conversion)

// Boolean conversion
console.log(Boolean(1));       // true
console.log(Boolean(0));       // false
console.log(Boolean(""));      // false
console.log(Boolean("hello")); // true
console.log(!!"hello");        // true (double negation)

Type Coercion

JavaScript may automatically convert types when an operation involves mismatched types: 

console.log("5" + 2);      // "52" (number converted to string)
console.log("5" - 2);      // 3 (string converted to number)
console.log("5" * "2");    // 10 (strings converted to numbers)
console.log(true + 1);     // 2 (true is converted to 1)
console.log(false + 1);    // 1 (false is converted to 0)
