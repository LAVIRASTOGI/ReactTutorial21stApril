For...of Loops

The for...of statement was introduced in ES6 and creates a loop iterating over iterable objects like arrays, strings, maps, sets, and more.

Basic Syntax

for (const element of iterable) {
  // code to execute for each element
}

Example with Arrays

const fruits = ['apple', 'banana', 'orange'];

for (const fruit of fruits) {
  console.log(fruit);
}
// Output:
// apple
// banana
// orange

Example with Strings

const text = 'Hello';

for (const char of text) {
  console.log(char);
}
// Output:
// H
// e
// l
// l
// o

Example with Maps

const map = new Map([
  ['name', 'John'],
  ['age', 30]
]);

for (const [key, value] of map) {
  console.log(`${key}: ${value}`);
}
// Output:
// name: John
// age: 30

For...of vs. For...in

for...of iterates over the values in an iterable. for...in iterates over the enumerable property names of an object. 

const arr = ['a', 'b', 'c'];

// for...of gets values
for (const value of arr) {
  console.log(value); // 'a', 'b', 'c'
}

// for...in gets indices/keys
for (const key in arr) {
  console.log(key); // '0', '1', '2'
}

Benefits of For...of Loops

  • Cleaner syntax than traditional for loops
  • Works with any iterable object
  • Doesn't need index counters
  • Can use with break, continue, and return statements

Try it yourself

Open your browser console (F12) and try these examples:

// Iterate over array
const numbers = [1, 2, 3, 4, 5];
for (const num of numbers) {
  console.log(num * 2);
}

// Iterate with destructuring
const people = [
  { name: 'Alice', age: 25 },
  { name: 'Bob', age: 30 }
];
for (const {name, age} of people) {
  console.log(`${name} is ${age} years old`);
}
