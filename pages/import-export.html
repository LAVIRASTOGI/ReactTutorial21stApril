Import and Export in JavaScript

JavaScript modules allow you to break up your code into separate files. This makes it easier to maintain a code-base.

ES modules use the import and export statements to share code between files.

Named Exports

You can export multiple named values from a module:

// moduleA.js
export const name = "Alice";
export const age = 30;
export function sayHello() {
  return `Hello, my name is ${name}`;
}

Then import them in another file:

// main.js
import { name, age, sayHello } from './moduleA.js';

console.log(name); // "Alice"
console.log(age); // 30
console.log(sayHello()); // "Hello, my name is Alice"

Default Exports

Each module can have one default export:

// moduleB.js
const person = {
  name: "Bob",
  age: 25
};

export default person;

Import a default export without using curly braces:

// main.js
import person from './moduleB.js';

console.log(person.name); // "Bob"

Mixing Default and Named Exports

You can combine default and named exports in a single module:

// moduleC.js
export const API_KEY = "abc123";
export const BASE_URL = "https://api.example.com";

const config = {
  timeout: 5000,
  retries: 3
};

export default config;

And import them together:

// main.js
import config, { API_KEY, BASE_URL } from './moduleC.js';

console.log(API_KEY); // "abc123"
console.log(config.timeout); // 5000

Renaming Imports

You can rename imports using the as keyword:

// main.js
import { name as userName, age as userAge } from './moduleA.js';

console.log(userName); // "Alice"
console.log(userAge); // 30

Importing Everything

You can import all exports from a module as a namespace object:

// main.js
import * as moduleA from './moduleA.js';

console.log(moduleA.name); // "Alice"
console.log(moduleA.age); // 30
console.log(moduleA.sayHello()); // "Hello, my name is Alice"

Try It Yourself

Let's create a simple example using our existing util.js file:

// util.js
export const apikey = "123hfdjkse45nkllw";
export const apikey1 = "123hfdjkse45nkllwdd";
const apikey2 = "123hfdjkse45nkllw";

export default "123hfdjkse45nkllw";

Now let's import these values:

// import-export-demo.js
// Default import
import defaultApiKey from '../util.js';
console.log("Default API Key:", defaultApiKey);

// Named imports
import { apikey, apikey1 } from '../util.js';
console.log("Named API Key:", apikey);
console.log("Named API Key 1:", apikey1);

// Import with rename
import { apikey as mainApiKey } from '../util.js';
console.log("Renamed API Key:", mainApiKey);

// Import all as namespace
import * as apiUtils from '../util.js';
console.log("All API Keys:", apiUtils);
// Note: Default export is available as apiUtils.default
