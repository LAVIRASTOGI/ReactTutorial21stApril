Array Methods in JavaScript

JavaScript arrays come with many built-in methods that allow you to perform common operations like adding, removing, or transforming elements.

Understanding these methods is essential for efficient array manipulation and for writing clean, readable code.

Creating Arrays

There are several ways to create arrays in JavaScript:

// Array literal (most common)
const fruits = ['apple', 'banana', 'orange'];

// Array constructor
const numbers = new Array(1, 2, 3, 4, 5);

// Creating an array with predefined length
const emptyArray = new Array(5); // Creates array with 5 empty slots

// Array.from() - creates array from array-like or iterable object
const arrayFromString = Array.from('hello'); // ['h', 'e', 'l', 'l', 'o']
const arrayFromSet = Array.from(new Set([1, 2, 2, 3])); // [1, 2, 3]

// Array.of() - creates an array from its arguments
const arrayOf = Array.of(1, 2, 3); // [1, 2, 3]

Basic Array Methods

Methods for adding and removing elements:

const fruits = ['apple', 'banana', 'orange'];

// push() - adds elements to the end, returns new length
const newLength = fruits.push('grape', 'kiwi');
console.log(fruits); // ['apple', 'banana', 'orange', 'grape', 'kiwi']
console.log(newLength); // 5

// pop() - removes the last element, returns the removed element
const lastFruit = fruits.pop();
console.log(fruits); // ['apple', 'banana', 'orange', 'grape']
console.log(lastFruit); // 'kiwi'

// unshift() - adds elements to the beginning, returns new length
const newLength2 = fruits.unshift('mango', 'pineapple');
console.log(fruits); // ['mango', 'pineapple', 'apple', 'banana', 'orange', 'grape']
console.log(newLength2); // 6

// shift() - removes the first element, returns the removed element
const firstFruit = fruits.shift();
console.log(fruits); // ['pineapple', 'apple', 'banana', 'orange', 'grape']
console.log(firstFruit); // 'mango'

// splice() - changes array by removing/replacing existing elements and/or adding new elements
const removed = fruits.splice(2, 1, 'cherry', 'peach'); // Start at index 2, remove 1 item, add 'cherry' and 'peach'
console.log(fruits); // ['pineapple', 'apple', 'cherry', 'peach', 'orange', 'grape']
console.log(removed); // ['banana']

// slice() - returns a shallow copy of a portion of an array (doesn't modify original)
const sliced = fruits.slice(1, 4); // From index 1 to 3 (4 not included)
console.log(sliced); // ['apple', 'cherry', 'peach']
console.log(fruits); // ['pineapple', 'apple', 'cherry', 'peach', 'orange', 'grape'] (unchanged)

Array Transformation Methods

Methods that transform arrays (these don't modify the original array): 

const numbers = [1, 2, 3, 4, 5];

// map() - creates a new array by applying a function to each element
const doubled = numbers.map(num => num * 2);
console.log(doubled); // [2, 4, 6, 8, 10]

// filter() - creates a new array with elements that pass a test
const evenNumbers = numbers.filter(num => num % 2 === 0);
console.log(evenNumbers); // [2, 4]

// reduce() - reduces array to a single value by applying a function from left to right
const sum = numbers.reduce((total, num) => total + num, 0);
console.log(sum); // 15 (1+2+3+4+5)

// reduceRight() - like reduce(), but works from right to left
const concatenated = ['a', 'b', 'c'].reduceRight((result, char) => result + char, '');
console.log(concatenated); // 'cba'

// flat() - creates a new array with all sub-array elements flattened
const nestedArray = [1, [2, 3], [4, [5, 6]]];
console.log(nestedArray.flat()); // [1, 2, 3, 4, [5, 6]] (default depth: 1)
console.log(nestedArray.flat(2)); // [1, 2, 3, 4, 5, 6] (depth: 2)

// flatMap() - combination of map() followed by flat() with depth 1
const sentences = ['Hello world', 'How are you?'];
const words = sentences.flatMap(sentence => sentence.split(' '));
console.log(words); // ['Hello', 'world', 'How', 'are', 'you?']

Array Search and Iteration Methods

Methods for finding elements and iterating through arrays:

const fruits = ['apple', 'banana', 'cherry', 'apple', 'date'];

// indexOf() - returns the first index at which a given element is found
console.log(fruits.indexOf('apple')); // 0
console.log(fruits.indexOf('apple', 1)); // 3 (start from index 1)
console.log(fruits.indexOf('grape')); // -1 (not found)

// lastIndexOf() - returns the last index at which a given element is found
console.log(fruits.lastIndexOf('apple')); // 3

// includes() - determines whether an array includes a certain element
console.log(fruits.includes('banana')); // true
console.log(fruits.includes('kiwi')); // false

// find() - returns the first element that satisfies the provided testing function
const numbers = [5, 12, 8, 130, 44];
const found = numbers.find(num => num > 10);
console.log(found); // 12

// findIndex() - returns the index of the first element that satisfies the provided testing function
const index = numbers.findIndex(num => num > 10);
console.log(index); // 1

// forEach() - executes a provided function once for each array element
fruits.forEach((fruit, index) => {
  console.log(`${index}: ${fruit}`);
});
// 0: apple
// 1: banana
// 2: cherry
// 3: apple
// 4: date

// some() - tests whether at least one element passes the provided function
const hasLongName = fruits.some(fruit => fruit.length > 5);
console.log(hasLongName); // true (banana has 6 characters)

// every() - tests whether all elements pass the provided function
const allShortNames = fruits.every(fruit => fruit.length < 10);
console.log(allShortNames); // true

Array Ordering Methods

Methods for changing the order of elements:

// sort() - sorts the elements of an array in place
const fruits = ['banana', 'cherry', 'apple', 'date'];
fruits.sort();
console.log(fruits); // ['apple', 'banana', 'cherry', 'date']

// Sorting numbers requires a compare function
const numbers = [40, 1, 5, 200];
numbers.sort(); // Default sort is lexicographic (string-based)
console.log(numbers); // [1, 200, 40, 5] (incorrect numeric order)

// Correct numeric sort
numbers.sort((a, b) => a - b);
console.log(numbers); // [1, 5, 40, 200]

// reverse() - reverses the order of the elements in an array
fruits.reverse();
console.log(fruits); // ['date', 'cherry', 'banana', 'apple']

Array Conversion Methods

Methods for converting arrays to other forms:

const fruits = ['apple', 'banana', 'cherry'];

// join() - joins all elements of an array into a string
const fruitString = fruits.join(', ');
console.log(fruitString); // 'apple, banana, cherry'

// toString() - returns a string representing the array
console.log(fruits.toString()); // 'apple,banana,cherry'

// Converting an array to a Set (to remove duplicates)
const numbersWithDuplicates = [1, 2, 2, 3, 4, 4, 5];
const uniqueNumbers = [...new Set(numbersWithDuplicates)];
console.log(uniqueNumbers); // [1, 2, 3, 4, 5]

Array Information Methods

Methods for getting information about arrays:

const array = [1, 2, 3, 4, 5];

// length - property that returns the number of elements
console.log(array.length); // 5

// Array.isArray() - determines whether the passed value is an array
console.log(Array.isArray(array)); // true
console.log(Array.isArray('not an array')); // false

// Checking if an array is empty
const isEmpty = array.length === 0;
console.log(isEmpty); // false
